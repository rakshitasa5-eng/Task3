### E-Commerce Sales Analysis

Dataset Description

     -->The dataset includes one merged table containing:
  
     -->Customer Details (ID, name, email)
  
     -->Product Details (product ID, product name, category, price)
  
     -->Order Information (order ID, order date, total amount, quantity)

Each row represents one order placed by a customer.

## Insights Generated From the Dataset

This project explores multiple analytical insights, such as:

✔ Total revenue generated by each product category

Helps understand which category contributes most to business.

✔ Identification of the highest-spending customer

Useful for customer segmentation and loyalty strategies.

✔ Average quantity ordered per product

Shows product demand patterns.

✔ Top-selling product based on quantity sold

Indicates the most popular product among customers.
Etc...
